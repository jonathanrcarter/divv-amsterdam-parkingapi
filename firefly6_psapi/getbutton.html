Get the button

By Copying the following code in your website you can use the great Park Shark Amsterdam API

In this example the Latitude is 52.359523 and the Longitude is 4.889753

<LINK REL="stylesheet" TYPE="text/css" HREF="http://api.parkshark.nl/button/psapi_frame_client.css">
<script type="text/javascript" src="http://api.parkshark.nl/button/psapi_frame_client.js"></script>
<a href="javascript:parksharkapi.pop(52.359523,4.889753);"><img src='http://api.parkshark.nl/button/images/parkshark_trigger.png' border='0'></a>

This code will add all the functionality you need to have.
