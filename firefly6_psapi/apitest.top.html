Dienst Infrastructuur Verkeer en Vervoer, Amsterdam (DIVV Amsterdam)

DIVV Amsterdam Parking API

The goal of the API is to provide app developers with the available parking data from Amsterdam and bring it together with some business logic which makes it immediately useable in apps.

Audience

Any level of developer who know how to work with APIs and Data.

Development

This was developed by Glimworm IT BV in Amsterdam. Glimworm are also responsible for ParkShark Amsterdam.

Support

The API is hosted and supported by Glimworm IT BV

Documentation

Documentation can be found at Documentaion site

Ownership

The DIVV Amsterdam commissioned and paid for the API to be created and they own the API

Formats

The data is returned in json format
